## 🌟 Overview

This repository contains a curated collection of **bank logos in SVG format** from **India and international markets**. All logos are optimized for web use, scalable, and maintained in high quality. Ideal for:

- **Financial applications** and fintech platforms
- **Banking software** and mobile apps
- **Payment gateways** and e-commerce websites
- **Educational projects** and research
- **Web developers** and UI/UX designers

## 🚀 Usage

### Direct Download
Simply clone this repository and use the SVG files:

```bash
git clone https://github.com/yourusername/all-bank-logo.git
cd all-bank-logo
```

### HTML Usage
```html
<img src="src/assets/bank/indian-bank/hdfc.svg" alt="HDFC Bank Logo" width="100">
<img src="src/assets/bank/international-bank/chase.svg" alt="JPMorgan Chase Logo" width="100">
```

### React/Vue Usage
```javascript
import hdfcLogo from './assets/bank/indian-bank/hdfc.svg'
import chaseLogo from './assets/bank/international-bank/chase.svg'
```

### CSS Background
```css
.bank-logo {
  background-image: url('src/assets/bank/indian-bank/sbi.svg');
  background-size: contain;
  background-repeat: no-repeat;
}
```
